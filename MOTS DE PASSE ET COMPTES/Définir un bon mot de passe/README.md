# Définir un bon mot de passe

Un mot de passe robuste doit respecter plusieurs critères essentiels :

- **La longueur** : 14 caractères minimum.
- **L'imprévisibilité** : c'est-à-dire ne contenir aucune information liée à votre vie personnelle.
- **La complexité** : il doit contenir des chiffres ainsi que des caractères spéciaux.
- **L'unicité** : un mot de passe différent pour chaque compte. L'usage d'un **gestionnaire de mots de passe** devient ici quasiment indispensable.
- **Le secret** : il ne doit pas avoir été compromis dans une fuite de données ou être présent dans une liste de mots de passe connue (comme _rockyou.txt_).

> Pour vérifier si votre mot de passe est compromis, vous pouvez utiliser [Have I Been Pwned](https://haveibeenpwned.com/) ou des outils comme [DeHashed](https://dehashed.com/).

## Pourquoi la longueur est-elle cruciale ?

La longueur est le facteur de sécurité le plus important, car c'est la meilleure défense contre les attaques par **brute force**. Il est souvent conseillé d'utiliser une **passphrase** (une phrase complète servant de mot de passe).

Pour tester la robustesse d'un format (attention : ne testez jamais votre _vrai_ mot de passe, mais un modèle similaire), vous pouvez consulter : [How Secure Is My Password?](https://www.security.org/how-secure-is-my-password/).

## Bon à savoir

Il n'est **plus recommandé** de changer régulièrement de mot de passe. Forcer un changement fréquent incite souvent les utilisateurs à adopter des comportements à risque (mots de passe trop simples ou répétitifs).
